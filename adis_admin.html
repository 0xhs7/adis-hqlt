🔗 Connect Google Sheets

One-time setup. Once connected, all student submissions flow into your Google Sheet automatically.

📋 Setup Steps

Create a new Google Sheet. Name it ADIS HQLT Responses. 4 tabs will be created automatically when the first submission arrives.

In the sheet, click Extensions → Apps Script. Delete all code in the editor.

Open the file adis_apps_script.gs (downloaded with this admin panel). Copy all its contents and paste into the Apps Script editor. Click Save (Ctrl + S).

Click Deploy → New Deployment. Set Type → Web App. Set Execute as → Me. Set Who has access → Anyone. Click Deploy and grant access when prompted.

Copy the Web App URL that appears (looks like https://script.google.com/macros/s/.../exec).

Paste it in the field below and click Save & Connect.

🔑 Your Apps Script Web App URL

This single URL handles all submissions and data reads. Keep it private.
